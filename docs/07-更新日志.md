## [3.0.0] - 2025-11-29

### 新增
- **微服务架构**: Gateway + Workers 分离
- **服务注册发现**: ServiceRegistry
- **负载均衡**: round_robin, least_load, random
- **限流系统**: TokenBucket + SlidingWindowCounter
- **WebSocket**: 实时状态推送
- **Web 页面**: /status, /admin, /playground

### 变更
- 代码重构到 `voice-clone-tts/src/`
- 配置统一到 `config.yaml`

## 版本号规则

`MAJOR.MINOR.PATCH`

- **MAJOR**: 不兼容的 API 变更
- **MINOR**: 向后兼容的功能新增
- **PATCH**: 向后兼容的问题修复
